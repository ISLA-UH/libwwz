libwwz demo 2: Smartphone Accelerometer Data of an Earthquake¶

This notebook will demonstrate a basic example of using libwwz specified in the following list.

  • Load Pickled Data
  • Use libwwz to compute the WWZ and WWA of unevenly sampled data using the 'octave' method
  • Plot the computed WWZ and WWA using 'octave' method
  • Reformat plot to include ticks in datetime

Data used in this demo is taken from redvox.io website: http://redvox.io/@/b589

We got a brief look at the data, let's do some transforms...¶

The octave method uses generalized CQT frequency bands for the analyzing frequencies.

In [8]:
# Select the size for the newly generated timestamps (tau), it should be less than the length of the given data.
# This will also limit the minimum possible frequency
ntau = 300

# Set the target frequency and the low and high frequency bounds that contain our signal of interest (target)
frequency_target = 2.5
frequency_low = 0.5
frequency_high = 7.5

# Set the logarithmatic scale base and band order for the octave bands
log_scale_base = 10**(3/10)     # G3
band_order = 3                  # 1/3 octave bands 

# Set the override to False (Setting to True will ignore the low and high frequency limitations)
override = False

# gather the frequency parameters into a list
frequency_parameters = [frequency_target, frequency_low, frequency_high, band_order, log_scale_base, override]

# We will then select the decay constant for our analyzing wavelet (negligible at < 0.2)
decay_constant = 0.0125


# Finally, we select wether to run with parallelization (recommend True)
parallel = True
In [9]:
# Now to get the WWZ/WWA from each xyz-sensor (This took about 5 minutes on a beefy-laptop)
# It can take longer or shorter depending on the resolution and computing power...

wwt_x = libwwz.wwt(timestamps=timestamps.values,
                   magnitudes=data['x'].values,
                   time_divisions=ntau,
                   freq_params=frequency_parameters,
                   decay_constant=decay_constant,
                   method='octave',
                   parallel=parallel)

wwt_y = libwwz.wwt(timestamps=timestamps.values,
                   magnitudes=data['y'].values,
                   time_divisions=ntau,
                   freq_params=frequency_parameters,
                   decay_constant=decay_constant,
                   method='octave',
                   parallel=parallel)

wwt_z = libwwz.wwt(timestamps=timestamps.values,
                   magnitudes=data['z'].values,
                   time_divisions=ntau,
                   freq_params=frequency_parameters,
                   decay_constant=decay_constant,
                   method='octave',
                   parallel=parallel)
